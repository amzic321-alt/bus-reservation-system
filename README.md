# Bus Ticket Reservation System
Group 7 - Programming Fundamentals Final Project

## How to Compile
gcc bus_reservation.c -o bus_reservation

## How to Run
./bus_reservation   (Mac/Linux)
bus_reservation.exe (Windows)

## Features
- View seat map
- Book a seat
- Cancel a booking
- Search passenger
- Print ticket
- Save/load bookings from file
